Como crear tu propio modulo

En Itsuki, un modulo es un archivo en modules/ que exporta funciones, clases o constantes con la palabra exportar.

1) Crea el archivo del modulo

# modules/mate.suki
exportar sea PI = 3.14159

exportar funcion sumar(a, b) {
  retornar a + b
}

sea secreto = 123  # no exportado

2) Importalo desde tu app

importar "modules/mate" como mate

print(mate.PI)
print(mate.sumar(10, 20))

3) Recomendaciones

  • Exporta solo API publica; deja lo interno sin exportar.
  • Usa nombres claros para funciones y constantes.
  • Evita ejecutar logica pesada al cargar el modulo.
  • Si el modulo crece, usa un init.suki para reexportar.

Estructura sugerida

modules/
  mi_lib/
    init.suki
    strings.suki
    numeros.suki

Bonus: modulo nativo C

Para alto rendimiento, crea un modulo en C compilado como libreria compartida y cargalo con importar "modules/mi_modulo". 

# Windows (.dll)
gcc -shared -O3 -Wall -I./src/headers modules/mi_modulo.c -o modules/mi_modulo.dll -lm

# Linux (.so)
gcc -shared -fPIC -O3 -Wall -I./src/headers modules/mi_modulo.c -o modules/mi_modulo.so -lm

Tambien puedes usar el Makefile de modules/ para compilar rapido: 

cd modules
make        # todos los modulos C
make math   # solo math.c

Ver guia completa: Modulos Nativos C.
