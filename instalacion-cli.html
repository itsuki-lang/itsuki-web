Instalacion y CLI

Podras ver como instalar itsuki

Instalacion rapida

git clone github.com/Apolo1061/itsuki.git
cd itsuki
make

Comandos principales

Comando Descripcion
itsuki archivo.suki Ejecuta un script.
itsuki Abre REPL interactivo.
itsuki --help Muestra ayuda CLI.
itsuki --lsp Inicia el servidor LSP.
itsuki fmt archivo.suki Formatea codigo.
itsuki lint archivo.suki Linter del archivo.

Compilacion y salida

Modo Ejemplo Resultado
Nativo itsuki app.suki -o app Genera binario compilando bundle C.
Codigo C itsuki app.suki -c app.c Exporta C embebido.
Bytecode itsuki app.suki -b app.sukiby Genera bytecode.

Flags utiles

  • --debug: activa debug y step mode de la VM.
  • --profile: imprime perfil de tiempo por funcion.
  • --no-gc: activa modo manual de memoria.
  • --lint: lint antes de ejecutar.
  • --static, --strip, --keep-temp: opciones de build nativo.

REPL

Comandos internos del REPL:

  • :ayuda
  • :limpiar o :cls
  • :salir o :exit
