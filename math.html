Biblioteca math

Modulo nativo en C implementado en modules/math.c. El runtime carga modules/math.dll (Windows) o modules/math.so (Linux).

Importacion

importar "modules/math" como math

Constantes

Constante Valor
math.PI 3.141592653589793
math.TAU 6.283185307179586
math.E 2.718281828459045

Funciones exportadas

Funcion Firma Descripcion
math.clamp (x, min, max) Limita un valor entre min y max.
math.factorial (n) Factorial para enteros >= 0 (max 170).
math.mcd (a, b) Maximo comun divisor.
math.mcm (a, b) Minimo comun multiplo.
math.es_primo (n) Retorna true si n es primo.
math.fibonacci (n) Fibonacci para entero >= 0 (max 93).
math.suma (array) Suma elementos numericos de un array.
math.promedio (array) Promedio de un array no vacio.
math.min_lista (array) Minimo de una lista numerica.
math.max_lista (array) Maximo de una lista numerica.
math.mediana (array) Mediana de una lista numerica.
math.raiz (x) Raiz cuadrada (x debe ser >= 0).
math.varianza (array) Varianza de un array no vacio.
math.desviacion (array) Desviacion estandar de un array.
math.distancia2d (x1, y1, x2, y2) Distancia euclidiana en 2D.
math.rango (inicio, fin, [paso]) Genera array de inicio a fin (fin excluido).

Ejemplo rapido

importar "modules/math" como math

sea xs = [1, 2, 3, 4, 5]
print("PI = " + string(math.PI))
print("suma = " + string(math.suma(xs)))
print("promedio = " + string(math.promedio(xs)))
print("es primo 97? " + string(math.es_primo(97)))
print("rango = " + string(math.rango(0, 5, 1)))
