Decoradores

Itsuki permite decorar funciones con @nombre_decorador. Puedes encadenar varios decoradores y luego invocar la funcion normalmente. 

funcion log_llamada(func) {
  print("Decorador: Funcion '" + string(func) + "' fue decorada")
  retornar func
}

funcion medir_tiempo(func) {
  print("Decorador: Midiendo tiempo de ejecucion")
  retornar func
}

@log_llamada
funcion saludar(nombre) {
  print("Hola, " + nombre + "!")
}

@log_llamada
@medir_tiempo
funcion calcular(a, b) {
  retornar a + b
}

print("=== PRUEBA DE DECORADORES ===")
saludar("Maria")
sea resultado = calcular(10, 20)
print("Resultado: " + string(resultado))

Funciones de orden superior

Puedes usar callbacks con mapear, filtrar, cada y reducir. 

sea nums = [1,2,3,4,5]

funcion es_par(x) { retornar x % 2 == 0 }
funcion doble(x) { retornar x * 2 }
funcion sumar(a, b) { retornar a + b }

sea pares = filtrar(nums, es_par)
sea dobles = mapear(pares, doble)
sea total = reducir(dobles, sumar, 0)
print(total)
