Biblioteca sistem

Modulo nativo para interactuar con el sistema operativo, estilo OS de Python.

Importacion

importar "modules/sistem" como os

Constantes

Constante Descripcion
os.SEP Separador de directorios del sistema.
os.PATHSEP Separador de variables tipo PATH.
os.EXE_EXT Extension de ejecutable (ej. .exe en Windows).

Funciones

Funcion Firma Descripcion
os.version () Version del modulo.
os.name () Nombre de plataforma: windows, linux o darwin.
os.pid () PID del proceso actual.
os.run (cmd) Ejecuta comando del sistema y retorna codigo.
os.spawn (cmd) Lanza proceso en segundo plano y retorna ID interno.
os.wait (proc_id, [timeout_ms]) Espera un proceso lanzado con spawn.
os.try_wait (proc_id) Consulta no bloqueante del estado de un proceso.
os.kill (proc_id) Intenta terminar un proceso lanzado con spawn.
os.process_destroy (proc_id) Libera recursos internos del proceso gestionado.
os.cwd () Directorio actual.
os.chdir (path) Cambia directorio actual.
os.listdir ([path]) Lista nombres en un directorio.
os.scandir ([path]) Lista entradas con metadatos basicos (nombre/tipo).
os.walk (path) Recorre arbol de carpetas y retorna una lista de nodos.
os.mkdir (path) Crea un directorio.
os.makedirs (path) Crea arbol de directorios.
os.rmdir (path) Elimina directorio vacio.
os.remove (path) Elimina archivo.
os.rename (src, dst) Renombra/mueve ruta.
os.copy (src, dst) Copia archivo de src a dst.
os.move (src, dst) Mueve archivo/directorio (alias semantico de rename).
os.exists (path) True si existe.
os.is_file (path) True si es archivo.
os.is_dir (path) True si es directorio.
os.stat_size (path) Tamano de archivo en bytes.
os.chmod (path, mode) Cambia permisos del path.
os.chown (path, uid, gid) Cambia owner/grupo (solo POSIX).
os.utime (path, atime, mtime) Actualiza timestamps de acceso/modificacion.
os.symlink (src, dst) Crea enlace simbolico.
os.readlink (path) Lee destino de enlace simbolico (solo POSIX).
os.access (path, [mode]) Verifica acceso a una ruta.
os.getenv (key, [default]) Lee variable de entorno.
os.setenv (key, value, [overwrite]) Setea variable de entorno.
os.unsetenv (key) Elimina variable de entorno.
os.path_join (part1, part2, ...) Une segmentos de ruta.
os.basename (path) Nombre base de una ruta.
os.dirname (path) Directorio padre de una ruta.
os.splitext (path) Retorna [root, ext].
os.normpath (path) Normaliza separadores y segmentos ./...
os.abspath (path) Ruta absoluta.
os.temp_dir () Directorio temporal del sistema.
os.home_dir () Directorio home del usuario.
os.getuid () UID del proceso (solo POSIX).
os.getgid () GID del proceso (solo POSIX).
os.umask (mask) Cambia mascara de permisos y retorna la anterior (solo POSIX).

Compatibilidad

API Windows Linux/macOS
chown, readlink, getuid, getgid, umask No disponible Disponible
symlink Disponible segun permisos/politica del sistema Disponible

Ejemplo rapido

importar "modules/sistem" como os

print(os.name())
print(os.cwd())
print(os.path_join("modules", "sistem" + os.EXE_EXT))
print(os.normpath("./a/../b//c.txt"))

os.setenv("ITSUKI_X", "123")
print(os.getenv("ITSUKI_X", "none"))
os.unsetenv("ITSUKI_X")
