Biblioteca socket

Esta biblioteca se usa como modulo Itsuki y esta implementada en modules/socket.suki. El modulo envuelve funciones internas del runtime.

Importacion correcta

importar "modules/socket" como net

Constantes exportadas

Constante Valor Uso
net.AF_INET 2 Familia IPv4.
net.SOCK_STREAM 1 Socket TCP.
net.SOCK_DGRAM 2 Socket UDP.
net.SOCK_RAW 3 Socket RAW.

Funciones del modulo

Funcion Firma Descripcion
net.socket (familia, tipo) Crea socket. Si pasas nulo, usa AF_INET + SOCK_STREAM.
net.bind (sock, host, port) Asocia socket local a host/puerto.
net.listen (sock, backlog) Activa escucha TCP. Si backlog es nulo, usa 5.
net.accept (sock) Acepta cliente y retorna [socketCliente, ip].
net.connect (sock, host, port) Conecta cliente TCP.
net.send (sock, data) Envia texto por socket.
net.recv (sock, buflen) Recibe texto. Si buflen es nulo, usa 4096.
net.sendto (sock, data, host, port) Envio UDP.
net.recvfrom (sock, buflen) Recepcion UDP, retorna [mensaje, ip].
net.close (sock) Cierra el socket.

Alias disponibles

El modulo tambien exporta alias: leer, escribir, cerrar, conectar, unir, escuchar, aceptar.

RAW: estado real

Si, Itsuki expone RAW por net.SOCK_RAW y lo pasa a socket(AF_INET, SOCK_RAW, 0) en C.

  • En muchos sistemas RAW requiere permisos admin/root.
  • Sin permisos, el runtime lanza error Fallo socket.
  • La API actual no expone opciones avanzadas de RAW (setsockopt/protocolo especifico/cabeceras custom). 
importar "modules/socket" como net

sea r = net.socket(net.AF_INET, net.SOCK_RAW)
net.close(r)

Ejemplo TCP cliente (validado con tests)

importar "modules/socket" como net

sea s = net.socket(net.AF_INET, net.SOCK_STREAM)
net.connect(s, "example.com", 80)

sea req = "GET / HTTP/1.1\r\nHost: example.com\r\nConnection: close\r\n\r\n"
net.send(s, req)

sea respuesta = net.recv(s, 1024)
print(subcadena(respuesta, 0, 120))
net.close(s)

Ejemplo UDP local (validado con tests)

importar "modules/socket" como net

sea s = net.socket(net.AF_INET, net.SOCK_DGRAM)
net.bind(s, "127.0.0.1", 9999)
net.sendto(s, "Hola UDP", "127.0.0.1", 9999)

sea res = net.recvfrom(s, 1024)
sea datos = res[0]
sea addr = res[1]
print("Recibido: " + datos + " de " + addr)

net.close(s)
