Biblioteca socket

Esta biblioteca se usa como modulo Itsuki y esta implementada en modules/socket.suki. El modulo envuelve funciones internas del runtime.

Importacion correcta

importar "modules/socket" como net

Constantes exportadas

Constante Valor Uso
net.AF_INET 2 Familia IPv4.
net.SOCK_STREAM 1 Socket TCP.
net.SOCK_DGRAM 2 Socket UDP.
net.SOCK_RAW 3 Socket RAW.
net.IPPROTO_ICMP 1 Protocolo ICMP.
net.IPPROTO_TCP 6 Protocolo TCP.
net.IPPROTO_UDP 17 Protocolo UDP.
net.IPPROTO_RAW 255 Protocolo RAW.
net.SOL_SOCKET 1 Nivel de opciones de socket.
net.SO_REUSEADDR 2 Reutilizar direccion.
net.SO_BROADCAST 6 Permite broadcast.
net.SO_RCVTIMEO 20 Timeout de recepcion.
net.SO_SNDTIMEO 21 Timeout de envio.
net.TCP_NODELAY 1 Desactiva Nagle.
net.SHUT_RD 0 Shutdown lectura.
net.SHUT_WR 1 Shutdown escritura.
net.SHUT_RDWR 2 Shutdown lectura y escritura.

Funciones del modulo

Funcion Firma Descripcion
net.socket (familia, tipo) Crea socket. Si pasas nulo, usa AF_INET + SOCK_STREAM.
net.raw_socket (proto, familia) Crea socket RAW (solo Linux).
net.bind (sock, host, port) Asocia socket local a host/puerto.
net.listen (sock, backlog) Activa escucha TCP. Si backlog es nulo, usa 5.
net.accept (sock) Acepta cliente y retorna [socketCliente, ip, puerto].
net.connect (sock, host, port) Conecta cliente TCP.
net.send (sock, data) Envia texto por socket.
net.recv (sock, buflen) Recibe texto. Si buflen es nulo, usa 4096.
net.sendto (sock, data, host, port) Envio UDP.
net.recvfrom (sock, buflen) Recepcion UDP, retorna [mensaje, ip, puerto].
net.setsockopt (sock, level, opt, value) Configura opciones del socket.
net.getsockopt (sock, level, opt) Lee opciones del socket.
net.set_timeout (sock, recv_ms, send_ms) Timeouts de recv/send en milisegundos.
net.shutdown (sock, how) Cierra lectura/escritura segun how.
net.set_hdrincl (sock, enabled) Activa IP_HDRINCL (solo Linux).
net.sendto_raw (sock, bytes, host, port) Envio RAW con bytes (solo Linux).
net.close (sock) Cierra el socket.

Alias disponibles

El modulo tambien exporta alias: leer, escribir, cerrar, conectar, unir, escuchar, aceptar.

RAW: estado real

Itsuki expone RAW por net.raw_socket y permite enviar paquetes con cabeceras propias.

  • En muchos sistemas RAW requiere permisos admin/root.
  • Sin permisos, el runtime lanza error Fallo socket.
  • En Windows, RAW esta deshabilitado por seguridad.
importar "modules/socket" como net

sea r = net.raw_socket(net.IPPROTO_RAW)
net.close(r)

Ejemplo TCP cliente (validado con tests)

importar "modules/socket" como net

sea s = net.socket(net.AF_INET, net.SOCK_STREAM)
net.connect(s, "example.com", 80)

sea req = "GET / HTTP/1.1\r\nHost: example.com\r\nConnection: close\r\n\r\n"
net.send(s, req)

sea respuesta = net.recv(s, 1024)
print(subcadena(respuesta, 0, 120))
net.close(s)

Ejemplo UDP local (validado con tests)

importar "modules/socket" como net

sea s = net.socket(net.AF_INET, net.SOCK_DGRAM)
net.bind(s, "127.0.0.1", 9999)
net.sendto(s, "Hola UDP", "127.0.0.1", 9999)

sea res = net.recvfrom(s, 1024)
sea datos = res[0]
sea addr = res[1]
sea port = res[2]
print("Recibido: " + datos + " de " + addr + ":" + string(port))

net.close(s)
